## 📌 Как использовать

1. Скопируйте ссылку выше
2. Откройте один из плееров
3. Вставьте ссылку и наслаждайтесь просмотром

### 📱 SS IPTV
- Откройте приложение
- Перейдите во вкладку **«Плейлисты»**
- Нажмите **«Добавить плейлист»**
- Вставьте ссылку → **«Сохранить»**

### 💻 VLC
- Откройте VLC
- **Медиа** → **Открыть URL**
- Вставьте ссылку → **«Воспроизвести»**

### 📲 IPTV Smarters
- Откройте приложение
- Выберите **«Загрузить плейлист по ссылке»**
- Вставьте ссылку → **«Добавить пользователя»**
